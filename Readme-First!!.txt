Building:

How to run NEWTON.PAS   - run ./CrossReference NEWTON.PAS

Bad Test Application    - run ./CrossReference badtest.PAS

Using Test Application:
Once you run the Application with test.PAS you will see the cases running.
If you run bad test.PAS you will see the cases crashing.
